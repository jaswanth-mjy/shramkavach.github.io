## Troubleshooting

### "Sitemap XML not parseable"
- Check sitemap.xml syntax using online validator
- Verify all URLs use HTTPS
- Ensure XML declaration at top: `<?xml version="1.0" encoding="UTF-8"?>`

### "URL not indexed"
- Use Mobile-Friendly Test: https://search.google.com/test/mobile-friendly
- Check if robots.txt blocks the page
- Verify no noindex tag (unless intentional)
- Check Fetch & Render to see what Google sees

### "Low value content" flag persists
- Add more content to pages (500+ more words)
- Increase unique value/unique analysis
- Add user testimonials or reviews
- Include recent statistics/data
- Re-request indexing after edits

### "Crawl errors" appearing
- Check for 404 errors (broken links)
- Verify redirects are 301 (permanent)
- Check for SSL/HTTPS issues
- Ensure mobile rendering correct
