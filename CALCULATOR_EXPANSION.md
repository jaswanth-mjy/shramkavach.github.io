## üí° Key Formulas Implemented

### **Financial Formulas:**
- **EMI**: `P √ó r √ó (1+r)^n / ((1+r)^n - 1)` where r = monthly interest
- **Compound Interest (FD)**: `P √ó (1 + r/4)^(4√óyears)` (quarterly compounding)
- **SIP Maturity**: `Monthly √ó (((1+r)^n - 1) / r) √ó (1+r)`
- **Break-Even Units**: `Fixed Costs / (Selling Price - Variable Cost)`
- **ROI**: `(Final Value - Investment) / Investment √ó 100`

### **Tax Formulas:**
- **Old Regime**: Standard deduction ‚Çπ50,000 + slabs (0%, 5%, 20%, 30%)
- **New Regime**: No deductions + lower slabs (0%, 5%, 10%, 15%, 20%, 30%)
- **GST Exclusive**: `Amount √ó (1 + GST Rate)`
- **GST Inclusive**: `Amount / (1 + GST Rate)` to extract base
- **TDS**: `Payment Amount √ó TDS Rate` (20% if no PAN)

### **Labour Law Formulas:**
- **Gratuity**: `(Basic Pay / 26) √ó 15 √ó Years` (minimum 1 year for fixed-term)
